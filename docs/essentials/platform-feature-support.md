# <a name="platform-support"></a>플랫폼 지원

Xamarin.Essentials는 다음과 같은 플랫폼 및 운영 체제를 지원합니다.

| 플랫폼 | 버전 |
| --- | --- |
| Android | 4.4(API 19) 이상 |
| iOS |10.0 이상 |
| Tizen | 4.0 이상 |
| tvOS | 10.0 이상 |
| watchOS | 4.0 이상 |
| UWP | 10.0.16299.0 이상 |
| macOS | 10.12.6(Sierra) 이상 |

> [!NOTE]
>
> * Tizen은 Samsung 개발 팀에서 공식적으로 지원합니다.
> * tvOS 및 watchOS의 API 범위는 제한되어 있습니다. 자세한 내용은 기능 가이드를 참조하세요.
> * macOS 지원은 미리 보기 상태입니다.
