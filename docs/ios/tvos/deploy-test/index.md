## <a name="app-distributioniostvosdeploy-testapp-distributionindexmd"></a>[앱 배포](~/ios/tvos/deploy-test/app-distribution/index.md)

이 문서에서는 다음과 같은 다양 한 방법을 통해 배포용 tvOS 앱을 구성, 빌드 및 게시 하는 방법을 보여 줍니다.

- Apple TV 앱 스토어 배포.

- 임시 배포.
