# <a name="can-i-use-a-mac-vm-with-xamarin"></a>Xamarin에서 Mac VM을 사용할 수 있나요? 

## <a name="mac-hardware"></a>Mac 하드웨어
Yes; Apple의 약관에 따라에 주의 해야 합니다.

## <a name="non-mac-hardware"></a>Mac이 아닌 하드웨어
아니요. Apple의 약관이이 사용을 허용 하지 않습니다.
