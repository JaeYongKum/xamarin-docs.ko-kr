이 가이드에서는 Android 응용 프로그램의 위치 인식을 소개 하 고, Android Location Service API를 사용 하 여 사용자의 위치를 가져오는 방법 및 Google Location Services API에서 사용할 수 있는 퓨즈 위치 공급자를 보여 줍니다.
