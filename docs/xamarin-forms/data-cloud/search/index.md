# <a name="searching-data-in-the-cloud"></a>클라우드에서 데이터 검색

## <a name="searching-data-with-azure-searchazure-searchmd"></a>[Azure Search를 사용 하 여 데이터를 검색합니다.](azure-search.md)

이 문서에서는 Microsoft Azure Search 라이브러리를 사용 하 여 Xamarin.Forms 응용 프로그램에 Azure Search를 통합 하는 방법에 설명 합니다.
