이 연습에서는 MonoGame 유니버설 Windows 플랫폼 (UWP) 프로젝트 생성 및 로드 하는 콘텐츠를 설명 합니다. UWP 앱은 데스크톱, 태블릿, Windows Phone 및 Xbox One을 포함 하 여 모든 Windows 10 장치에서 실행할 수 있습니다.
